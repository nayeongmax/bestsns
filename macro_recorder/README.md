# 게임 매크로 레코더 (Game Macro Recorder)

키보드 및 마우스 입력을 녹화하고 반복 재생하는 매크로 프로그램입니다.
아이온2 등 게임에서 반복 작업(몹 사냥, 골드 수급 등)을 자동화할 수 있습니다.

## 설치 방법

### 1. Python 설치
- [Python 3.8 이상](https://www.python.org/downloads/) 설치
- 설치 시 **"Add Python to PATH"** 체크 필수

### 2. 필수 라이브러리 설치
```bash
cd macro_recorder
pip install -r requirements.txt
```

## 사용 방법

### 실행
```bash
python macro_recorder.py
```

> **중요**: 게임 내 입력이 정상 작동하려면 **관리자 권한**으로 실행하세요.
> 명령 프롬프트(cmd)를 관리자로 실행한 후 위 명령을 입력합니다.

### 단축키

| 키 | 기능 |
|-----|------|
| **F9** | 녹화 시작 (기본 60초) |
| **F10** | 반복 재생 시작/중지 |
| **F12** | 프로그램 종료 |

### 사용 순서

1. **프로그램 실행** - `python macro_recorder.py`
2. **게임 창으로 전환**
3. **F9 누르기** - 녹화가 시작됩니다 (60초)
4. **원하는 동작 수행** - 방향키 이동, 마우스 클릭, 스킬 사용 등
5. **녹화 자동 완료** - 60초 후 자동으로 녹화가 끝납니다
6. **F10 누르기** - 녹화한 동작이 무한 반복 재생됩니다
7. **F10 다시 누르기** - 재생 중지
8. **F12** - 프로그램 종료

### 녹화 데이터 저장

- 녹화가 완료되면 `macro_data.json` 파일로 자동 저장됩니다
- 다음에 프로그램을 실행하면 저장된 매크로가 자동으로 로드됩니다
- 바로 F10을 눌러 이전에 녹화한 매크로를 재생할 수 있습니다

## 설정 변경

`config.json` 파일을 만들어 설정을 변경할 수 있습니다:

```json
{
  "record_duration": 120
}
```

| 설정 | 기본값 | 설명 |
|------|--------|------|
| `record_duration` | 60 | 녹화 시간 (초) |

## 아이온2 활용 팁

1. **몹 사냥 매크로**
   - 사냥할 위치로 이동
   - F9로 녹화 시작
   - 주변을 돌며 몹을 공격 (방향키 + 마우스 클릭 + 스킬키)
   - 1분간 반복 루트를 돌며 녹화
   - F10으로 무한 반복 재생

2. **녹화 팁**
   - 시작 지점과 끝 지점을 같은 위치로 맞추면 자연스러운 반복이 됩니다
   - 너무 빠른 입력보다는 안정적인 패턴이 좋습니다
   - 게임 해상도와 창 위치를 고정한 상태에서 녹화하세요

## 주의사항

- **Windows 전용**: 게임 입력 호환성을 위해 Windows SendInput API를 사용합니다
- **관리자 권한**: 게임에 입력을 전달하려면 관리자 권한이 필요합니다
- **게임 약관**: 매크로 사용은 게임 이용약관에 위배될 수 있습니다. 사용에 따른 책임은 본인에게 있습니다
- **해상도 고정**: 녹화 시와 재생 시 모니터 해상도 및 게임 창 위치가 동일해야 합니다

## 문제 해결

| 문제 | 해결 방법 |
|------|-----------|
| 게임에서 입력이 안 됨 | 관리자 권한으로 실행 |
| 마우스 위치가 다름 | 해상도/창 위치를 녹화 시와 동일하게 설정 |
| pip 설치 오류 | `python -m pip install -r requirements.txt` 시도 |
| 녹화 데이터가 없다고 나옴 | F9로 먼저 녹화 필요 |
